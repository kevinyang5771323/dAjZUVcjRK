# 民宿平台管理系统 java95

## 项目概述

民宿平台管理系统是基于SpringBoot开发的，采用前后端分离的技术架构，前端使用Vue.js框架，后端使用SpringBoot框架，数据库采用MySQL。

## 系统角色

- 普通游客（用户）
- 管理员

## 技术栈

- 后端：SpringBoot
- 前端：Vue.js
- 数据库：MySQL

## 功能模块

### 系统首页

- 浏览民宿酒店类型
- 支持通过时间搜索、民宿名称、民宿类型进行搜索

### 房间简介

- 查看民宿简介、评论

### 预定民宿

- 在民宿详情页面进行预订

### 用户登录/注册

- 用户登录后才能进行预订

### 后台管理系统

#### 可视化统计模块

- 预订访问统计可视化界面

#### 用户管理模块

- 用户管理
- 管理员工管理
- 员工管理

#### 系统菜单管理模块

- 菜单管理
- 角色管理
- 日志管理
- 文件管理

#### 民宿管理模块

- 房源管理
- 房源图片管理

#### 订单管理模块

- 订单管理
- 财务管理

#### 业务办理模块

- 民宿预订/入住管理
- 预订处理
- 入住管理/退房管理

## 运行环境

- 后端：Java 1.8+
- 前端：Node.js
- 数据库：MySQL 5.7+
